## 💡 关键经验总结

### 必须记住的 3 点

1. **特殊字符 = 自定义 SCSS**
   - 遇到 `[` `]` `/` 立即创建自定义类

2. **动态类名 = 完整类名**
   - 永远不要拼接字符串
   - 使用方法返回或完整三元表达式

3. **标签转换 = view/text/image**
   - 不要忘记 image 的 mode 属性
